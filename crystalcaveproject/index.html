Crystal Cave

2D platformer game made in godot 4.3

The game challenges you to time every jump, dash, and climb with precision while avoiding enemies, spikes, and falling off the map.

Project development story

This was the first working game that I made and it was OSAO Kaukovainio Technical School project I made during my first year studying game programming and IT. The main goal of the project was to learn the basics of game engines and game development.
